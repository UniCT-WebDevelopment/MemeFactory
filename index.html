  • (Something) About Us:
Il sito sfrutta la libreria FFMPEG per unire due video insieme. In input abbiamo il file con estensione MP4 da caricare tramite il form nel server. Il video viene rigenerato con lo stesso timestamp del video in coda e successivamente viene unito al file depositato sul server. La concatenazione avviene tramite un file di testo che contiene i video da unire nell'ordine desiderato. Il file di testo viene generato ogni volta che viene inviato un nuovo file sul server. Al termine delle operazioni ffmpeg, viene lanciato un redirect verso il modulo di download. I File rimangono in memoria e non vengono cancellati al termine del download, verrano cancellati da una routine(non implementata) ogni ora. Funzione totalmente async
